üîß Setup Instructions

Step 1: Configure Discord Application

In your Discord Developer Portal:

  • Go to OAuth2 ‚Üí Redirects
  • Add redirect URL: https://renegaderaider.wtf/discord/callback
  • For development: http://localhost:5000/discord/callback

Step 2: Environment Variables

Make sure these are set in your environment:

DISCORD_CLIENT_ID=1393354181536120966 DISCORD_CLIENT_SECRET=your_client_secret_here

Step 3: Add Login Button to Your Site

Use this JavaScript to generate the login URL:

const CLIENT_ID = '1393354181536120966'; const REDIRECT_URI = encodeURIComponent('https://renegaderaider.wtf/discord/callback'); const SCOPES = 'identify'; const discordLoginURL = `https://discord.com/oauth2/authorize?client_id=${CLIENT_ID}&redirect_uri=${REDIRECT_URI}&response_type=code&scope=${SCOPES}`; // Use in your HTML: // <a href="${discordLoginURL}">Login with Discord</a>

Step 4: Test the Flow

The OAuth flow works as follows:

  1. User clicks "Login with Discord" ‚Üí Redirects to Discord
  2. User authorizes ‚Üí Discord redirects to /discord/callback
  3. Your server exchanges code for access token
  4. Server fetches user info and stores it
  5. User is redirected back to your site with stored data

üîç How It Works

When a user completes Discord OAuth:

  • User data is displayed on a success page
  • Data is stored in localStorage as 'discordUser'
  • User is auto-redirected to your site after 5 seconds
  • Your frontend can access the stored user data
// Access stored Discord user data in your frontend: const discordUser = JSON.parse(localStorage.getItem('discordUser') || '{}'); console.log('Discord user:', discordUser);
